# Atelectasis Detection using Chest X-Ray Images

## Overview
This project aims to detect Atelectasis from Chest X-Ray images using Machine Learning and Computer Vision techniques. The system processes medical images, extracts relevant features, and performs automated classification to assist in diagnosis.

## Features
- Chest X-Ray image preprocessing
- Region of Interest (ROI) extraction
- Feature engineering
- Machine Learning-based classification
- Automated prediction interface

## Technologies Used
- Python
- OpenCV
- NumPy
- Machine Learning
- HTML/CSS

## Dataset
Chest X-Ray medical image dataset containing Atelectasis and non-Atelectasis cases.

## Results
- Processed over 1,000 medical images
- Achieved approximately 64.5% classification accuracy

## Project Structure
- data_preparation.py
- train_model.py
- predict_pneumonia.py
- model_architecture.py
- evaluate_and_report.py
